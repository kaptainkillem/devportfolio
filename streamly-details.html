Streamly – A Full-Stack Netflix Clone

Streamly is a Netflix-inspired streaming web application built to deliver a smooth and engaging user experience. The platform features user authentication, an advanced search function, dynamic thumbnails, and a personalized watchlist. Designed with a responsive UI, Streamly ensures seamless access across all devices.

  • Website: none
  • Completed: 2024

Technologies Used & Implementation
✅ Front-End Development React.js – Used for building dynamic, reusable UI components for a smooth user experience. HTML & CSS – Provides structure and styling for a clean and modern interface. Bootstrap – Ensures a responsive design across different screen sizes.
✅ Back-End Development Node.js & Express.js – Handles authentication, API requests, and backend logic. PHP – Manages session handling and backend security.
✅ Database Management MySQL & MongoDB – Used for storing user profiles, watch history, and media content.
✅ Authentication & Security Cybersecurity Techniques (Kali Linux) – Used to test and secure user data against vulnerabilities.
✅ Additional Features & Enhancements JavaScript – Implements interactive elements such as search, UI responsiveness, and real-time updates. C++ & C# – Used for optimizing performance and exploring AI-driven content recommendations.

This project showcases my expertise in full-stack development, database management, and security implementation while highlighting my ability to build scalable and user-friendly applications.
